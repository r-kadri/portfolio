# Ryan Kadri - Portfolio

This is my personal website | ongoing.

## Required

* PHP 8.1
* Composer
* Symfony CLI
* Docker
* Docker-compose

You can check the requirements by running this Symfony CLI command :
```bash
symfony check:requirements
```

## Run

```bash
docker compose up -d
```

## Run tests

```bash
php bin/phpunit --testdox
```
